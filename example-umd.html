Quick Start

Liquid Glass UMD

Use with a simple script tag. No build tools required.
Just pure, elegant glass for your web projects.

Drag me
Drag to interact 
<!-- 1. Include the styles -->
<link rel="stylesheet" href="liquid-glass.css">

<!-- 2. Include the UMD script -->
<script src="liquid-glass.umd.js"></script>

<!-- 3. Initialize -->
<script>
  var glass = new LiquidGlass('#glass', {
    draggable: true,
    surfaceType: 'convex_squircle',
    bezelWidth: 30
  });
</script>
